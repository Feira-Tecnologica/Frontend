Testar Atualização de Projeto

Clique no botão abaixo para testar a funcionalidade de edição:

Instruções para uso:

  1. Execute o arquivo migration_projeto_edicao.sql no seu banco de dados para adicionar os campos necessários
  2. Acesse a página editar_projeto.html
  3. Faça as alterações desejadas nos campos do projeto
  4. Clique em "SALVAR ALTERAÇÕES"
  5. O projeto será atualizado no banco de dados

Funcionalidades implementadas:

  • ✅ Edição do nome do projeto
  • ✅ Edição da descrição
  • ✅ Edição do objetivo
  • ✅ Edição da justificativa
  • ✅ Gerenciamento de integrantes (adicionar/remover)
  • ✅ Seleção de ODS (Objetivos de Desenvolvimento Sustentável)
  • ✅ Validação de campos obrigatórios
  • ✅ Feedback visual durante o salvamento
